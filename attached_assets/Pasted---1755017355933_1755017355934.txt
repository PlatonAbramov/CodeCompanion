Внизу рядом с "Заказчики" "Подрядчики" "Главная" "Сотрудники" нужно добавить меню-вкладку "Инструменты" 

Вкладка «Инструменты» 
Что видно:
По умолчанию — список «Сейчас в работе» (выданные и не возвращённые).
Табы/фильтры: Все, В наличии (на складе), В работе (у кого-то), Списанные (необязательный статус).
Поиск по названию инструмента и по имени/телефону человека.

Строка списка:
Название инструмента.
Статус: В наличии / У человека (Имя).
Дата/время последнего действия.
Цена (стоимость).
Иконка фото (если есть последняя фотография).
Кнопка действия: Выдать (если в наличии) / Вернуть (если у человека).

Кнопки:
Добавить инструмент.

2) Карточка инструмента
Поля (редактируемые при создании/правке):
Название инструмента (обязательное).
Инвентарный номер / внутренний код (необязательно, но уникальное, если заполнено).
Стоимость (валюта проекта; число ≥ 0).
Краткое описание (необязательно).
Блок текущего статуса:
Статус: В наличии / У человека (Имя, телефон).

Дата/время выдачи (если выдан, необязательно).
Фото при выдаче (миниатюра, открывается в полноразмер).
История движений (хронология):
Каждое событие: Выдано → Возвращено, с датами/временем(необязательно), кем оформлено, кому выдано (имя, телефон), фото выдачи и фото возврата.

Возможность открыть фото.

Действия:
Выдать (если в наличии) → открывает форму выдачи.
Вернуть (если у человека) → открывает форму возврата.
Редактировать (для админа всегда; для пользователя — только карточку без изменения истории).
Списать (только админ; переводит в статус «Списанные», выдавать нельзя).

3) Форма «Выдать»
Цель: зафиксировать, кто берёт, когда и приложить фото инструмента на момент выдачи.
Поля:
Кому выдаём:
Имя (обязательное).
Телефон (обязательное, проверка формата +971-хх-ххх-хх-хх).
(Опция) выбрать из недавних — список последних 20 людей (имя+телефон).

Фото при выдаче (обязательное; нельзя завершить без фото).

Дата/время выдачи (заполняется автоматически текущими; можно поправить вручную админом).

Комментарий (необязательно).

Кнопки: Сохранить / Отмена.
Результат: инструмент переходит в статус «У человека». В истории создаётся запись «Выдано» с фото, временем и данными человека.

4) Форма «Вернуть»
Цель: зафиксировать возврат и состояние, приложить фото при возврате.

Поля:

Фото при возврате (обязательное).

Дата/время возврата (авто; можно поправить админом).

Комментарий (необязательно; например, «царапина на корпусе»).

Кнопки: Сохранить / Отмена.
Результат: инструмент переходит в статус «В наличии». В истории к текущему циклу добавляется «Возвращено» с фото и временем.

Минимальные правила и проверки
Обязательное фото: без фото нельзя завершить «Выдать» и «Вернуть».

Целостность операций: нельзя выдать инструмент, который уже «У человека»; нельзя вернуть «В наличии».

Обязательные поля при выдаче: Имя, Телефон, Фото.

Формат телефона: простая проверка (цифры + «+», «()», «-», пробелы). Сохранить нормализованное значение.

Время: фиксируется от системного; хранить в UTC с отображением в локальном.

Стоимость инструмента: не меняется задним числом при операциях, редактируется только в карточке.

Редактирование истории: пользователи не могут менять прошлые записи; админ может править комментарии/время и, при необходимости, заменить фото (с пометкой в истории «исправлено админом»).

Удаление: обычным пользователям нельзя; админ может удалить инструмент (только если нет открытой выдачи) и отдельные события (с отметкой в истории «удалено админом»).

История и аудит
В карточке инструмента хранятся все события.

Любое редактирование админом — фиксировать флаг и время/кто исправил.
Удаление событий/инструментов — мягкое (soft delete) с пометкой в аудите.

Импорт/экспорт (минимум)
Экспорт списка инструментов и истории в CSV (по фильтру и по дате).
Импорт начального списка инструментов из CSV (название, инв.номер, стоимость).

Производительность и ограничения
Команда до 20 человек, до ~1000 инструментов — операции и списки должны открываться <1 сек при нормальной сети.

Фото хранить с ограничением размера на загрузку (например, до 20 МБ); показывать превью, по клику — оригинал.
