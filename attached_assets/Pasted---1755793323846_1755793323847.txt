ТЗ: Раздел «Персонал» (видят только Администратор и Прораб)
Цель

Добавить пункт «Персонал» в нижней навигации, доступный только Администратору и Прорабу. Раздел содержит карточки сотрудников с фото, личными данными, документами (любой формат), зарплатой, стажем и специализацией. Реализовать подсветку сроков документов (жёлтый ≤30 дней, красный ≤14 дней/истёк).

Видимость и права (жёстко)

Администратор
Видит «Персонал». Может создавать/редактировать/удалять карточки, добавлять/удалять фото и документы, редактировать зарплату, дату начала работы, специализацию, любые поля.

Прораб
Видит «Персонал». Только просмотр карточек и документов. Ничего не меняет (без кнопок «создать/редактировать/удалить», без загрузки/удаления файлов).

Мастер
Не видит вкладку «Персонал» (пункт меню отсутствует; прямой доступ по URL — запрещён).

Заказчик
Не видит вкладку «Персонал» (пункт меню отсутствует; прямой доступ по URL — запрещён).

Проверка прав делается на уровне меню, страниц и API-запросов. Прямые ссылки и ручные запросы должны возвращать «Нет доступа».

Навигация

В нижней панели отображать пункт «Персонал» только Админу и Прорабу.

Клик → список карточек сотрудников (сетка/таблица), поиск и фильтры.

Список сотрудников (карточки)

Отображать на каждой карточке:

Фото (миниатюра).

ФИО.

Emirates ID (номер/серия).

Даты по Emirates ID: номер, дата выдачи, дата окончания (компактно в одну строку).

Специализация.

Стаж (расчёт от даты начала, формат «X лет Y мес.»).

Зарплата (AED, формат «AED 4,500»).

Индикатор срока (по наихудшему статусу среди ключевых документов: норм/жёлтый/красный).

Подсветка

Жёлтый — до истечения ≤30 дней.

Красный — до истечения ≤14 дней или документ истёк.

Документы для контроля минимум: Emirates ID, паспорт, виза, контракт (если есть срок).

Карточка сотрудника (внутри)
Блок «Профиль»

Фото (большое).

Админ: кнопки «Добавить фото» / «Удалить фото».

Прораб: только просмотр.

Поля: Фамилия, Имя, Отчество (опц.), дата рождения (опц.), телефон, email (опц.), специализация, дата начала работы, стаж (расчётное), зарплата (AED), статус (Активен/Уволен/Отпуск — опц.).

Админ: редактирует и сохраняет.

Прораб: только просмотр.

Сводный индикатор сроков документов (бейджи по ключевым документам).

Вкладка «Документы»

Админ: «Загрузить документ» (любой формат), указать тип (Emirates ID/паспорт/виза/контракт/прочее), номер/серия, дата выдачи/окончания, комментарий; удалить документ.

Прораб: только просмотр списка документов, скачивание/просмотр файлов.

Превью изображений; для PDF/офиса — иконка + имя файла. Клик по изображению → модалка с большим фото, стрелки для перелистывания, закрыть.

Валидации и форматы

Валюта — AED везде.

Даты — единый формат (например, DD.MM.YYYY).

Номер/серия — строка (с ведущими нулями).

Ограничения по размеру/типу файлов — понятные ошибки при нарушении.

Удаление фото/документа — подтверждение.

Поиск и фильтры

Поиск: ФИО, Emirates ID, паспорт.

Фильтры: специализация, статус сотрудника, статус документов (норм/≤30/≤14/истёк).

Сортировки: по ФИО, стажу, ближайшей дате окончания Emirates ID.

История изменений (если включена в системе)

Логировать по сотруднику:

Создание/удаление карточки (кто, когда).

Изменения полей профиля (без вывода значения зарплаты — фиксируем факт изменения и кто).

Добавление/удаление фото/документа (тип, даты).

Изменение дат/номеров документов.

Безопасность и доступ

Пункт меню скрывать для Мастера и Заказчика.

Прямой доступ по URL для Мастера/Заказчика → запрет (страница «Нет доступа»).

Для Прораба все действия «только чтение». Попытки изменить/загрузить/удалить → запрет.

Критерии приёмки

Пункт «Персонал» в нижнем меню видят только Админ и Прораб.

Мастер и Заказчик не видят пункт меню; прямые ссылки на «Персонал» для них недоступны.

Список карточек отображает фото, ФИО, Emirates ID (номер/серия), даты выдачи/окончания, специализацию, стаж, зарплату, индикатор срока.

Подсветка работает: жёлтый ≤30 дней, красный ≤14/истёк (проверено на тестовых датах). Учитывается «самый критичный» документ.

Карточка сотрудника:

Админ — может создавать/редактировать/удалять, загружать/удалять фото и документы.

Прораб — только просмотр, скачивание/просмотр файлов, без кнопок изменения.

Документы любого формата загружаются, корректно отображаются/скачиваются; изображения открываются в модалке с листанием.

Стаж считается автоматически от даты начала; формат отображения корректный.

Поиск/фильтры/сортировки работают.

История изменений фиксирует операции согласно разделу «История изменений».

Никакой другой функционал не нарушен; производительность вкладки ≤1 сек до видимого контента.

Чек-лист для разработчиков

 Скрыть «Персонал» из меню для Мастера и Заказчика; запретить прямой доступ.

 Права: Админ — полный CRUD; Прораб — read-only.

 Список карточек: поля + индикатор сроков.

 Подсветка по датам документов (жёлтый/красный) с ежедневным перерасчётом и при изменениях.

 Карточка сотрудника: Профиль (CRUD у Админа), Документы (загрузка/удаление только у Админа).

 Превью изображений, модалка просмотра, листание.

 Поиск/фильтры/сортировки.

 Валидации форматов/размеров файлов; понятные ошибки.

 Логи изменений по сотруднику.

 Регресс: убедиться, что для Мастера/Заказчика раздел нигде не «просвечивает».
